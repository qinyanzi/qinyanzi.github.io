使用GitHub Pages + Hexo搭建blog

Hexo 本质上是一个 静态网站生成器，学习HTML后可以明白。大概就是将你github上的文件换个方式来进行访问，但比github在一定程度上甚至不如github。不能进行文件的上传，界面简陋，但简单并且一切都要从基础开始吗？我时候终会想这个达不到我的要求就不去尝试，正确但非完全正确。

hexo官网地址查看文档

前置需求

安装 Node.js，使用最新版本即可。安装 Git 会简单的git，不会也行

账号准备

GitHub 账号并创建一个仓库，名字必须是：你的git用户名.github.io

安装 Hexo

找个你想存放博客的文件夹，比如 D:/hexo-blog。打开终端（Windows 可以用 PowerShell / CMD）

初始化博客：

本地启动博客预览

**本地查看:**打开浏览器访问 http://localhost:400

配置 Hexo 部署到 GitHub

安装部署插件：
编辑 _config.yml 文件

（在 blog 根目录,就是前面初始化所使用的文件名），找到 deploy:

生成并部署

在 blog 文件夹执行（这个是你自己创建的文件中）：

在执行hexo deploy:fatal: detected dubious ownership in repository at ‘E:/LiFe/LinBlog/blog/.deploy_git’

解决
